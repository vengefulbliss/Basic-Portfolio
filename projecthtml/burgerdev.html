Welcome to Eat-da-burger

Eat-da-burger is a simple full stack application incorperating many different coding languages, and a database to make the app fully functional.

the application is pretty basic. you order a burger, the order is placed into the database, when you "devour" the burger it is taken from one table in the database and placed in another.

this project was reletively easy to develope. the difficulty mainly came from incorperating the database to make a fullstack application!

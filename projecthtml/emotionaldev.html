Welcome to emotional-control.

emotional-control is an alpha stage application made with the intent to customize a users experience around the their very own personality.
By using facial recognition we will be able to detect how a user responds to their searched content. eventually helping the browser know exactly what you mean by vague search inputs. (ex. simply typing funny has many different meanings... unless the browser knows exactly what you mean by how you respond visually.)

currently the application allows you to search for content. you can see content that matches your mood or simply browse.
eventually the application will act as an emotional ballance tool, populating calming content if you are anxious, happy if you are sad and so on.

I make sure to picture each element of a website before the coding process begins
